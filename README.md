# Chess-Quiz-Battle
Um jogo gamificado que combina xadrez reduzido com quiz de múltipla escolha. O objetivo é tornar o aprendizado mais dinâmico: sempre que uma peça for capturada no tabuleiro, ocorre uma “batalha de quiz”. Se o jogador acertar, mantém sua peça. Se errar, a peça é eliminada pelo oponente.
